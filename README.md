# 🎮 게임 플랫폼 및 엔진 소개

## 1. 콘솔 게임 (Console Game)
전용 게임기(하드웨어)를 TV나 모니터에 연결하여 즐기는 게임입니다.

* **대표 기기:** PlayStation, Xbox, Nintendo Switch 등
* **특징:**
    * 하드웨어에 최적화되어 **높은 그래픽 품질과 안정적인 성능**을 제공합니다.
    * 패드(컨트롤러)를 사용한 조작감이 뛰어납니다.
    * 대형 자본이 투입된 'AAA급 게임'이 주로 출시됩니다.

---

## 2. 웹 게임 (Web Game)
별도의 설치 없이 인터넷 브라우저(Chrome, Edge 등)에서 바로 실행 가능한 게임입니다.

* **주요 기술:** HTML5, WebGL, WebAssembly 등
* **특징:**
    * **접근성이 매우 높으며** 기기 사양의 제약을 덜 받습니다.
    * 링크 공유만으로 다른 사용자와 쉽게 함께 즐길 수 있습니다.
    * 최근에는 3D 그래픽 기술의 발전으로 고퀄리티 게임도 가능해졌습니다.

---

## 3. 유니티 게임 (Unity Game)
세계적으로 가장 대중적인 상용 게임 엔진인 'Unity'로 개발된 게임입니다.

* **주요 언어:** C# (C Sharp)
* **특징:**
    * **멀티플랫폼 지원:** 한 번의 개발로 PC, 모바일, 콘솔, VR/AR 등 다양한 기기로 배포가 가능합니다.
    * 방대한 에셋 스토어(Asset Store)를 통해 개발 리소스를 쉽게 구할 수 있습니다.
    * 인디 게임부터 대형 상업 게임(예: *원신, 어몽어스*)까지 폭넓게 사용됩니다.
